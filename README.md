# Bmy-CIRBP-TR8: A Novel Longevity Enhancer in the Bowhead Whale

This repository contains the raw sequence alignments and thermodynamic modeling data 
supporting the discovery of an 8-fold tandem repeat expansion in the CIRBP promoter 
of *Balaena mysticetus*.

## Discovery Summary
- [cite_start]**Target Gene:** CIRBP [cite: 5]
- [cite_start]**Locus:** -412 bp upstream of TSS [cite: 7, 9]
- [cite_start]**Mechanism:** SP1-mediated super-enhancer recruitment [cite: 11, 13]
